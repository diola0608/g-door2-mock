アプリ内のコード値と、OneRoster や文科省コードなどの外部規格との対応表を管理します。アプリ内の表示と書き出すデータが異なる場合に変換します。
sync_alt
変換の向き: エクスポート時は アプリ内の値 → 標準コード へ変換、インポート時は 標準コード → アプリ内の値 へ逆変換します。外部規格との連携は Composer → 外部システム の一方向が基本で、書き出したデータが外部側へ反映されます。実際の変換は取込・連携設定の変換ルールから呼び出されます。
