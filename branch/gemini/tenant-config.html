「氏名」列のスペース区切りを姓・名に分割し、「Female」などの性別表記を標準化します。

function transformRecord(row) { if (row['氏名']) { const parts = row['氏名'].split(' '); row['姓'] = parts[0]; if (parts.length > 1) row['名'] = parts[1]; } if (row['性別'] === 'Female' || row['性別'] === 'F') row['性別'] = '女性'; return row; }

sync
Workspaceへの一方向同期: ここで設定されたマッピングは、Google Workspace のディレクトリ属性への同期時にも適用されます（例：Google側の役職フィールドに「その他の職員」として連携）。同期は Composer → Workspace の一方向です。
