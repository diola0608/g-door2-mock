レコードの変換ルール
レコード変換（高度・JavaScript）

組織タイプや所属自治体による条件分岐、正規表現での分割など、DBに登録したJavaScriptで変換できます。例:「山田 花子（女性）」を 姓・名・性別 に振り分け、中学校では学年 3 → 9 に変換。

function transform(rec, ctx) { // "山田 花子（女性）" → 姓 / 名 / 性別 const m = rec.fullname.match(/^(\S+)\s+(\S+)（(.+)）$/); if (m) { rec.lastName = m[1]; rec.firstName = m[2]; rec.gender = m[3]; } // 中学校は学年を OneRoster 表記へ（3 → 9） if (ctx.schoolType === '中学校') rec.grade = Number(rec.grade) + 6; return rec; }
アプリ内の表示・登録・検索に使う値と、規格（OneRoster・文科省コード等）で入出力する値を相互変換します。例:「ICT支援員」はアプリ内ではそのまま扱い、エクスポート時は「その他の職員」へマッピングします。
