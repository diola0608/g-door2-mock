downloadエクスポート（外部への出力変換）

エクスポートはインポートの逆方向の変換を行います。アプリ内のデータを、外部システム／規格に合わせた形式に変換して出力します。

public外部コード規格への変換

アプリ内の表示名がそのまま外部に出力されるとは限りません。例: アプリ内「ICT支援員」→ MEXT/OneRosterで「99（その他の職員）」に出力。

規格マッピングはコード標準マッピングで管理します。

transformレコード単位の逆変換

姓と名が分かれているものを外部システムが連結で欲しがる場合などは、インポートとは逆の変換を JavaScript で定義できます。

変換ルールはインポート画面で登録したものを反転して適用します。

宛先規格適用される変換
OneRoster CSV出力 OneRoster 1.2 職種・所属コードを OneRoster の語彙に変換
